# ReFreshHeads
**ReFreshHeads is in no way affiliated with [Lidl](https://lidl.sk) or [Magneds](https://magneds.com/). Provided as-is.**

ReFreshHeads is a community revival of Lidl Fresh Heads, based on the Slovak version (v2.24), modified to work on modern Android without online dependencies.

## Features
- Removed Firebase and Pushwoosh  
- Fully* playable offline
- Supports modern Android (tested up to Android 16)  
- Smaller than the original APK  
- All marbles unlocked by default  

## Bugs
- Unable to save custom levels
- Some features originally requiring online services are unavailable
